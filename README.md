# Synapse Vercel Example

Example showing how to use [@pyrx/synapse](https://www.npmjs.com/package/@pyrx/synapse) with Vercel Serverless Functions.

## Setup

1. `npm install`
2. Set env vars in Vercel dashboard: `SYNAPSE_API_KEY`, `SYNAPSE_WORKSPACE_ID`
3. `vercel deploy`
